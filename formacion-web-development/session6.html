CSS moderno y responsive design

🎨 ¿Qué es CSS?

CSS = Cascading Style Sheets

Es el "diseñador de interiores" de tu web. Define cómo se ve el contenido HTML 🎭

CSS hace:

  • Colores, tipografías, tamaños
  • Layouts (distribución de elementos)
  • Animaciones y transiciones
  • Responsive design (adaptación a dispositivos)
💡 Analogía: Si HTML es la estructura de una casa, CSS es la decoración, pintura y diseño interior

📝 Sintaxis básica CSS

/* Estructura de una regla CSS */
selector {
  propiedad: valor;
  otra-propiedad: otro-valor;
}

/* Ejemplo real */
h1 {
  color: #06b6d4;
  font-size: 32px;
  text-align: center;
}

.boton {
  background-color: #06b6d4;
  color: white;
  padding: 10px 20px;
  border-radius: 5px;
}
Partes: Selector (qué) + Declaración (cómo)

🎯 Selectores CSS principales

Selector Sintaxis Selecciona
Elemento p { } Todos los <p>
Clase .boton { } class="boton"
ID #header { } id="header"
Descendiente div p { } <p> dentro de <div>
Hijo directo div > p { } <p> hijo directo de <div>

📦 Box Model - El modelo de caja

/* Cada elemento HTML es una caja con 4 partes */
.elemento {
  /* Contenido (width x height) */
  width: 300px;
  height: 200px;
  
  /* Padding (espacio interno) */
  padding: 20px;
  
  /* Border (borde) */
  border: 2px solid #06b6d4;
  
  /* Margin (espacio externo) */
  margin: 10px;
}

/* box-sizing: border-box hace el cálculo más intuitivo */
* {
  box-sizing: border-box;
}
📐 Cálculo: Ancho total = width + padding + border (con border-box)

🔲 Display - Tipos de elementos

/* Block: Ocupa todo el ancho disponible */
div {
  display: block;
}

/* Inline: Solo el ancho de su contenido */
span {
  display: inline;
}

/* Inline-block: Híbrido (mejor de ambos) */
.boton {
  display: inline-block;
}

/* None: Oculta el elemento */
.oculto {
  display: none;
}

🔄 Flexbox - Layout moderno

Flexbox facilita la distribución de elementos en una dimensión (fila o columna) 📏

Conceptos clave:

  • Container: Padre con display: flex
  • Items: Hijos directos del container
  • Main axis: Eje principal (horizontal)
  • Cross axis: Eje secundario (vertical)
💡 Perfecto para: Navegaciones, cards horizontales, centrar elementos

🎯 Flexbox - Propiedades del container

.container {
  display: flex;
  
  /* Dirección */
  flex-direction: row; /* row | column | row-reverse | column-reverse */
  
  /* Alineación horizontal (main axis) */
  justify-content: center; /* flex-start | flex-end | center | space-between | space-around */
  
  /* Alineación vertical (cross axis) */
  align-items: center; /* flex-start | flex-end | center | stretch */
  
  /* Salto de línea */
  flex-wrap: wrap; /* nowrap | wrap | wrap-reverse */
  
  /* Gap entre items */
  gap: 20px;
}

💻 Flexbox - Ejemplo: Navbar

/* HTML */
<nav class="navbar">
  <div class="logo">Logo</div>
  <ul class="menu">
    <li>Inicio</li>
    <li>Productos</li>
    <li>Contacto</li>
  </ul>
</nav>

/* CSS */
.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 20px;
  background: #06b6d4;
}

.menu {
  display: flex;
  gap: 30px;
  list-style: none;
}

📐 CSS Grid - Layout bidimensional

Grid permite crear layouts complejos en 2 dimensiones (filas Y columnas) 🎯

Conceptos clave:

  • Container: Padre con display: grid
  • Items: Hijos directos del container
  • Tracks: Filas y columnas
  • Cells: Intersecciones
💡 Perfecto para: Layouts de página completa, galerías, dashboards

🎯 CSS Grid - Propiedades básicas

.container {
  display: grid;
  
  /* Definir columnas */
  grid-template-columns: 200px 200px 200px;
  /* O con fr (fracciones) */
  grid-template-columns: 1fr 1fr 1fr;
  /* O con repeat */
  grid-template-columns: repeat(3, 1fr);
  
  /* Definir filas */
  grid-template-rows: 100px 200px;
  
  /* Gap entre items */
  gap: 20px;
  /* O separado */
  row-gap: 20px;
  column-gap: 30px;
}

💻 Grid - Ejemplo: Layout de página

.page-layout {
  display: grid;
  grid-template-columns: 250px 1fr;
  grid-template-rows: 80px 1fr 60px;
  grid-template-areas:
    "header header"
    "sidebar main"
    "footer footer";
  gap: 20px;
  height: 100vh;
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }
✨ Grid areas: Nombres visuales para posicionar elementos

⚖️ Flexbox vs Grid

🔄 Flexbox
  • ✅ Una dimensión (fila o columna)
  • ✅ Distribución de espacio
  • ✅ Alineación de items
  • ✅ Componentes pequeños

Usa para: Navegaciones, toolbars, cards en fila

📐 Grid
  • ✅ Dos dimensiones (filas y columnas)
  • ✅ Layouts complejos
  • ✅ Posicionamiento preciso
  • ✅ Páginas completas

Usa para: Layouts de página, galerías, dashboards

📱 Responsive Design

Responsive Design = Tu web se adapta a cualquier tamaño de pantalla 📱💻🖥️

Técnicas principales:

  • Meta viewport: Esencial en el HTML
  • Media queries: CSS condicional
  • Unidades relativas: %, rem, vw, vh
  • Mobile-first: Diseñar primero para móvil
💡 Breakpoints comunes: Mobile (320px+), Tablet (768px+), Desktop (1024px+)

📱 Media Queries

/* Mobile First: estilos base para móvil */
.container {
  width: 100%;
  padding: 10px;
}

/* Tablet (768px en adelante) */
@media (min-width: 768px) {
  .container {
    width: 750px;
    margin: 0 auto;
  }
}

/* Desktop (1024px en adelante) */
@media (min-width: 1024px) {
  .container {
    width: 1000px;
  }
}

/* Desktop grande (1280px en adelante) */
@media (min-width: 1280px) {
  .container {
    width: 1200px;
  }
}

💻 Ejemplo: Grid responsive

/* Mobile: 1 columna */
.grid {
  display: grid;
  grid-template-columns: 1fr;
  gap: 20px;
}

/* Tablet: 2 columnas */
@media (min-width: 768px) {
  .grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

/* Desktop: 3 columnas */
@media (min-width: 1024px) {
  .grid {
    grid-template-columns: repeat(3, 1fr);
  }
}

/* Desktop XL: 4 columnas */
@media (min-width: 1280px) {
  .grid {
    grid-template-columns: repeat(4, 1fr);
  }
}

🎨 Variables CSS (Custom Properties)

/* Definir variables en :root */
:root {
  --color-primary: #06b6d4;
  --color-secondary: #0891b2;
  --color-text: #333;
  --spacing-small: 10px;
  --spacing-medium: 20px;
  --spacing-large: 40px;
  --border-radius: 8px;
}

/* Usar variables */
.boton {
  background: var(--color-primary);
  color: white;
  padding: var(--spacing-medium);
  border-radius: var(--border-radius);
}

.card {
  border: 2px solid var(--color-primary);
  margin: var(--spacing-large);
}

📏 Unidades CSS modernas

Unidad Tipo Uso recomendado
px Absoluta Bordes, sombras
% Relativa Anchos, alturas
rem Relativa Font-size, spacing
em Relativa Padding proporcional
vw / vh Viewport Secciones fullscreen
